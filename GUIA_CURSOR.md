# 🚀 Guia Prático do Cursor

## O que é o Cursor?
O Cursor é um editor de código com IA integrada que te ajuda a programar de forma mais eficiente. Eu sou seu assistente de IA e posso:
- ✅ Escrever código para você
- ✅ Explicar código existente
- ✅ Corrigir erros
- ✅ Refatorar e melhorar código
- ✅ Criar projetos do zero
- ✅ Executar comandos no terminal

## 🎯 Como Me Usar Efetivamente

### 1. **Seja Específico**
❌ "Faça um site"
✅ "Crie um site de portfólio com HTML, CSS e JavaScript, com seções para sobre mim, projetos e contato"

### 2. **Pergunte Passo a Passo**
Em vez de pedir tudo de uma vez, vá construindo:
- "Crie a estrutura HTML básica"
- "Agora adicione o CSS para deixar bonito"
- "Adicione animações quando passar o mouse"

### 3. **Peça Explicações**
- "Explique como funciona essa função"
- "O que faz essa linha de código?"
- "Por que usar essa abordagem?"

## 📝 Comandos Úteis que Posso Executar

### Explorando Arquivos
- **Ver conteúdo de pastas**: "Mostre todos os arquivos da pasta X"
- **Ler arquivos**: "Mostre o conteúdo do arquivo Y"
- **Buscar código**: "Encontre onde a função Z é usada"

### Criando e Editando
- **Criar arquivo**: "Crie um arquivo chamado app.py"
- **Editar código**: "Adicione validação de email nesta função"
- **Refatorar**: "Melhore este código para ficar mais eficiente"

### Terminal
- **Instalar pacotes**: "Instale o pandas"
- **Executar código**: "Execute o arquivo main.py"
- **Git**: "Faça um commit com a mensagem 'feature: nova funcionalidade'"

## 💡 Dicas de Ouro

1. **Use o Contexto**: Eu consigo ver os arquivos abertos e entender o contexto do seu projeto

2. **Iteração Rápida**: Não precisa escrever tudo perfeito na primeira vez. Vá refinando:
   - "Crie uma função básica"
   - "Agora adicione tratamento de erros"
   - "Adicione documentação"

3. **Aprenda Fazendo**: Peça para eu criar exemplos práticos e depois modifique-os

4. **Não Tenha Medo de Errar**: Se algo não funcionar, me avise que eu corrijo!
